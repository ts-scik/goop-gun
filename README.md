## *goop-gun* is an unfinished prototype for an online multiplayer survival-horror shooter, made in the Godot Engine.

You can try a few different early release builds of this game.

***I recommend you try [v0.2](https://github.com/ts-scik/goop-gun/releases/tag/v0.2), and bring some friends!***

The builds are:
* [v0.3](https://github.com/ts-scik/goop-gun/releases/tag/v0.3) - the newest version, with:
  * multiplayer temporarily stripped out
  * initial reloading mechanics
  * refined gamepad handling / aiming mechanics
* [v0.2](https://github.com/ts-scik/goop-gun/releases/tag/v0.2) - an online multiplayer deathmatch test with:
  * a user-resizable infinitely-large procedurally-generated map, synchronized over the network
  * dynamically-generated network-synced physics objects
  * unique aiming mechanics, slightly more-refined
* [v0.1](https://github.com/ts-scik/goop-gun/releases/tag/v0.1) - an online multiplayer deathmatch test with:
  * a single-room map, with some fun nighttime lighting tests
  * a single network-synced physics barrel to play with
  * unique aiming mehanics
